# GX Coin Protocol

**A sovereign digital currency protocol for productive economics.**

GX Coin is a non-profit, public-utility blockchain protocol designed to capture, preserve, and transfer the value of human productivity — fairly and transparently.

## Core Design Principles

- **Fixed Supply:** 1.25 trillion GX units. No inflation. No expansion. Ever.
- **Gold-Anchored Reference:** 1 GX = 1 gram of gold at genesis. A universal unit of measure, not a redeemable claim.
- **0% Interest:** Capital is provided through profit-sharing, not interest-bearing debt.
- **Demurrage:** Progressive circulation incentive on large idle balances - money should move, not accumulate.
- **Proof of Authority:** Professional, accountable validators with immutable core monetary policies.
- **Free Distribution:** No ICO. No presale. No token sale. GX units are distributed directly to individuals through Genesis Distribution.

## Not a Speculative Token

GX Coin is not listed on any exchange. This is by design. Value is determined by purchasing power within the GX economy, not by speculative markets.
